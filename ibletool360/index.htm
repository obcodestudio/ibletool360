 Description:
     With the iBLETool360 Bluetooth-LE an exploration tool, you can explore the secrets within the exploration range of the BLE equipment, create a relationship with the BLE equipment, transmit and send data to the BLE equipment you designated for pairing. A message window lets you understand the communication status of the BLE equipment.
 Functions:
 Transmit function, Transmit data to the BLE equipment linked with designed service successfully.
 Data transmission with e-mail.
 Export a text file of your BLE data if you need a delimited text file to load into another program.
 Receive function, Receive data from the BLE equipment linked with designated service successfully.
 Scan function, Scan BLE equipments in the environment every 10 seconds automatically, and obtain device name, Identifier and RSSI data, which are transmitted using attached text file through Email.
 Setup function, BLE Name, Identifier and Service UUID data are stored by default.
 HID input function.
